# TheStreamDeck

A Raspberry Pi-based unified streaming audio appliance. One device, one amp input, instant source switching, album art on screen, and physical analogue VU meters.

## Sources

- Tidal HiFi Plus
- Plexamp (local library)
- BBC 6 Music
- Internet radio

## Features

- Touchscreen display with album artwork and transport controls
- Physical analogue VU meters showing real-time audio levels
- Instant source switching — tap to change from Tidal to BBC radio without touching the amplifier
- Single amp input eliminates multi-device cable management

## Platform

Raspberry Pi
